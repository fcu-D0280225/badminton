# 羽毛球預約系統 v2.0

支援館方、團主、臨打三種角色的預約管理系統。

## 角色功能

### 館方
- 紀錄被預約的時段且預約者資訊（包含付款紀錄）
- 被預約歷程列表，可用時間段、預約者姓名查詢
- 可以把自己的資訊寫在備註給團主跟臨打看

### 團主
- 紀錄已預約時間跟館方資料
- 紀錄已預約的臨打資訊
- 可以把自己的資訊寫在備註給臨打看

### 臨打
- 紀錄預約歷程
- 紀錄對館方跟團主的預約紀錄跟備註，並且可以用五分評分制打分數

## 技術架構

- 後端：NestJS + TypeORM + SQLite
- 前端：Node.js + Express + PWA
- 資料庫：SQLite

## 安裝和運行

1. 安裝依賴：
```bash
npm run install-all
```

2. 啟動開發伺服器：
```bash
npm run dev
```

3. 訪問應用：
- 前端： 
- 後端API：http://localhost:3000

## 手機安裝

詳細的手機安裝說明請參考 [MOBILE_SETUP.md](./MOBILE_SETUP.md)

**快速步驟：**
1. 確保手機和電腦在同一 WiFi
2. 啟動服務器後，會顯示網路 IP 地址
3. 在手機瀏覽器輸入該 IP 地址（例如：http://192.168.1.100:3001）
4. 選擇「添加到主畫面」即可安裝為 App

## 館方獨立 App

館方管理功能已分離成獨立的手機 App，使用不同的主題色（綠色）方便區分。

### 啟動館方 App

```bash
# 安裝依賴
cd venue-app
npm install

# 啟動館方 app（端口 3002）
npm run dev
```

### 各 App 端口

| App | 端口 | 說明 |
|-----|------|------|
| 後端 API | 3000 | NestJS 後端服務 |
| 主 App | 3001 | 團主、臨打功能（紫色主題） |
| 館方 App | 3002 | 館方管理功能（綠色主題） |

### 手機安裝

手機用戶可以分別安裝兩個獨立的 PWA：

1. **主 App**（http://IP:3001）
   - 團主管理
   - 臨打管理
   - 紫色主題

2. **館方 App**（http://IP:3002）
   - 館方管理
   - 預約日曆
   - 備註管理
   - 綠色主題
